# Zoom Clone

Socket IO를 사용한 Zoom Clone.
<ul>
  <li>Web RTC 방식으로 실시간 화상 채팅 구현</li>
  <li>Data Channel 방식으로 채팅 구현(예정)</li>
</ul>

## Setup & Run
```bash
# pnpm
pnpm install

# pnpm
pnpm dev
```
